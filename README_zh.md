# reggie_my

#### 介绍
本仓库是黑马的瑞吉外卖，本人进行了完善，并实现了用邮箱发送验证码。需要在配置文件中设置自己的邮箱账号和授权码。（因为邮箱发送验证码较慢，点击后需要等大约20秒）本程序没有进行大量测序可能有许多Bug，请自行修改^_^!
